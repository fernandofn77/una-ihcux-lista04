A heurística de Prevenção de Erros diz que o sistema deve evitar que erros aconteçam ou, quando acontecerem, lidar com eles de forma clara.

O try-catch ajuda nisso porque:

Impede que o sistema trave ao ocorrer um erro
Permite mostrar mensagens amigáveis ao usuário
Dá controle ao sistema para reagir corretamente ao problema

O try-catch não evita o erro em si, mas previne impactos negativos, tratando o erro de forma controlada e melhorando a experiência do usuário.
